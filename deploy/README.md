# Catalog Deployment

`deploy/authorized-catalog.json` is a sanitized example catalog that can be hosted on your own HTTPS static endpoint.

## Usage

1. Edit the JSON entries for your apps.
2. Upload the file to your HTTPS endpoint.
3. Paste the catalog URL in Macked Updater > Sources.
4. Search and add matching catalog entries.

Local testing can use a `file://` URL.

See `README.md` for the public catalog schema summary.
